CSS introduction

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML, or XHTML). CSS specifies how elements should be rendered on screen, on paper, in speech, or on other media. As one of the core languages of the open web, CSS is standardized across web browsers according to W3C specifications. Previously, the development of various parts of the CSS specification was done synchronously, allowing versioning of the latest recommendations, such as CSS1, CSS2.1, and CSS3. However, there will no longer be CSS3 or CSS4; now, it is simply referred to as "CSS" with individual CSS modules having version numbers.

After CSS 2.1, the scope of the specification increased significantly, and progress on different CSS modules started to vary so much that it became more effective to develop and release recommendations separately per module. Instead of versioning the CSS specification, W3C now periodically takes a snapshot of the latest stable state of the CSS specification and the progress of individual modules. CSS modules now have version numbers, or levels, such as CSS Color Module Level 5. This modular approach allows for more flexible development and quicker updates to various aspects of the CSS language, ensuring optimal compatibility and functionality across different devices and browsers.

CSS Basics

CSS (Cascading Style Sheets) is the code that styles web content. CSS basics walks through what you need to get started. We'll answer questions like: How do I make text red? How do I make content display at a certain location in the (webpage) layout? How do I decorate my webpage with background images and colors?

Like HTML, CSS is not a programming language. It's not a markup language either. CSS is a style sheet language. CSS is what you use to selectively style HTML elements. For example, this CSS selects paragraph text, setting the color to red:

                    p {
                    color: red;
                    }

Let's try it out! Using a text editor, paste the three lines of CSS (above) into a new file. Save the file as style.css in a directory named styles.

To make the code work, we still need to apply this CSS (above) to your HTML document. Otherwise, the styling won't change the appearance of the HTML. (If you haven't been following our project, pause here to read Dealing with files and HTML basics.)

  1. Open your index.html file. Paste the following line in the head (between the <head> and </head> tags): 
                                <link href="styles/style.css" rel="stylesheet" />
  2. Save index.html and load it in your browser. You should see something like this:
  3. Check if your paragraph text is red. If it is, congratulations! Your CSS is working.
  4. Now, let's add some more styles. In your style.css file, add the following code to change the background color and text alignment: 
                                body {
                            background-color: #f0f0f0;
                            text-align: center;
                            }
  5. Refresh your browser to see the changes. Your webpage should now have a light gray background and centered text.
CSS box model

The CSS box model module defines the height, width, margin, and padding properties, which along with border properties make up the CSS box model.

Every visible element on a webpage is a box laid out according to the visual formatting model. CSS properties define their size, position, and stacking level, with the box model properties (and others) defining the extrinsic size of each box, and the space around them.

Each box has a rectangular content area, inside which any text, images, and other content is displayed. The content may be surrounded by padding, a border, and a margin, on one or more sides. The padding is around the content, the border is around the padding, and the margin sits outside the border. The box model describes how these features — the content, padding, border, and margin — work together to create a box as displayed by CSS.

The CSS box model module defines physical (or "page relative") properties such as width and margin-top. Flow-relative properties such as inline-size and margin-block-start (which relate to text direction) are defined in Logical Properties and Values. The box model module is extended by the CSS box sizing module, which introduces the intrinsic size value and enables defining aspect ratios for elements that are auto-sized in at least one dimension.

CSS display

The display property in CSS is a fundamental aspect of web design, used to control the layout of elements on a web page. This property determines how an element is displayed and interacts with other elements. By understanding and utilizing the display property, developers can create more complex and responsive web designs. The display property can take on various values, each with distinct behaviors that influence the rendering of elements.

One of the most common values of the display property is block. An element with a display value of block takes up the full width available, starting on a new line. This is typical for elements like <div>, <h1>, and <p>. Block elements allow for margin and padding to be set, making them ideal for defining major sections of a webpage. In contrast, the inline value makes the element sit on the same line as its neighbors, only taking up as much width as necessary. Elements such as <span>, <a>, and <strong> are naturally inline, allowing for text styling without disrupting the flow of content.

Another crucial value is inline-block, which combines the properties of both inline and block elements. Elements with display: inline-block sit inline with other content but can have set widths, heights, margins, and paddings, similar to block elements. This value is particularly useful for creating horizontal navigation menus or side-by-side boxes that need precise control over dimensions and spacing.

Additionally, modern web design often employs flex and grid display values for more advanced layouts. The flex value is part of the Flexbox model, which simplifies the creation of flexible and responsive layouts. Flexbox allows elements to be aligned and distributed within a container, making it easier to build complex grid systems without using floats or positioning. The grid value, on the other hand, introduces a two-dimensional grid-based layout system, enabling developers to design web pages using rows and columns. This method provides unparalleled control over layout, making it possible to create intricate designs with relatively simple CSS rules.

CSS font styling

Font styling in CSS is a vital aspect of web design, as it significantly impacts the readability and aesthetic appeal of a website. Proper font styling ensures that text content is not only legible but also visually engaging, which can enhance user experience. CSS offers a variety of properties to style fonts, including font-family,font-size,font-weight,font-style,line-height,letter-spacing, and text-transform.

Font Family

The font-family property specifies the typeface to be used for the text. You can list multiple font families as a fallback mechanism in case the first choice is unavailable. Generic family names like serif, sans-serif, and monospace are also commonly used.

                    p {
                    font-family: "Arial", "Helvetica", sans-serif;
                    }

Font Size

The font-size property controls the size of the text. Sizes can be set using various units like pixels (px), em units (em), rem units (rem), percentages (%), and more. The use of relative units like em and rem is encouraged for better scalability and responsiveness.

                    h1  {
                    font-size: 2em;
                    }
                    p   {
                    font-size: 16px;
                    }

Font Weight

The font-weight property defines the thickness of the text. Common values include normal, bold, bolder, and lighter, or numeric values ranging from 100 to 900.

                    strong  {
                    font-weight: bold;
                    }
                    header   {
                    font-weight: 700;
                    }

Font Style

The font-style property is used to make text italic or oblique. The most common values are normal, italic, and oblique.

                    em  {
                    font-style: italic;
                    }

Line Height

The line-height property sets the amount of space between lines of text. This property is crucial for improving readability, especially in paragraphs.

                    p  {
                    line-height: 1.5;
                    }

Letter Spacing

The letter-spacing property controls the space between characters. Adjusting letter spacing can enhance the readability and aesthetic of the text.

                    h1  {
                    letter-spacing: 2px;
                    }

Text Transform

The text-transform property alters the case of the text. Common values include uppercase, lowercase, and capitalize.

                    header  {
                    text-transform: uppercase;
                    }

Combining Font Properties

For concise and efficient styling, you can combine several font properties using the shorthand font property. This shorthand can include font-style, font-variant, font-weight, font-size, line-height, and font-family.

                    p  {
                    font: italic small-caps bold 16px/1.5 "Georgia", serif;
                    }
